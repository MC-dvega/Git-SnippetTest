Changing the Cursor Speed

Cursor speed controls how fast the cursor moves when you move the Mood Mouse. To change the cursor speed, drag the slider.

Changing the Double-click Speed

Double-click speed controls how fast you must double-click for a response. To change the double-click speed, drag the slider.

Displaying Mood Mouse Trails

When mouse trails are enabled, they are displayed behind the cursor when you move the mouse. To display mouse trails, select the check box.

Setting the Orientation

Orientation adjusts the mouse movement according to the way you hold the Mood Mouse. To set the orientation, click Set Orientation and follow the steps in the wizard.
