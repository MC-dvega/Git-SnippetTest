Cursors

You can customize the cursor icons by using the default graphics included with Windows.

The icons you can change are:

  • Normal Select
  • Help Select
  • Working in Background
  • Busy
  • Precision Select
  • Text Select
  • Unavailable
  • Vertical Resize
  • Horizontal Resize
  • Diagonal Resize 1
  • Diagonal Resize 2
  • Move
  • Link Select

To change an icon:

  1. Select the icon that you want to change.
  2. Click Browse to open the Cursors folder.
  3. Select the graphic.
  4. Click Open.
