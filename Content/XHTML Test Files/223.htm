Adding a Moodicator Message

To add a Moodicator message:

  1. Click the drop-down arrow next to the message you want to change.
  2. Select New Message from the list. A new window is displayed.
  3. Type the message that you want to use.

    We suggest that you word your messages carefully, so as not to insult anyone who might use your system and Mood Mouse.

  4. Click OK.

Changing a Moodicator Message

To use a different Moodicator message:

  1. Click the drop-down arrow next to the message you want to change.

  2. Select the message from the list.

    Select New to add a new message to the list. When the dialog box is displayed, type the text of the new message.

  3. Click OK.
