Buttons

You can customize the left, right, thumb, and wheel buttons. The default actions are:

  • Left button: Click
  • Right button: Right-click
  • Thumb button: Display Moodicators
  • Wheel button: AutoScroll

To change a button action, click the appropriate drop-down arrow. The possible actions for each button are:

  • Click
  • Right-click
  • AutoScroll
  • Alt
  • Back
  • Backspace
  • Close
  • Copy (Ctrl + C)
  • Ctrl
  • Cut (Ctrl + X)
  • Delete
  • Double-click
  • Enter
  • Esc
  • F1 (Help)
  • F5 (Refresh)
  • Forward
  • Keystroke (combination)
  • Maximize
  • Minimize
  • Moodicators
  • None
  • Paste (Ctrl + V)
  • Print Screen
  • Redo
  • Run
  • Search
  • Select All
  • Shift
  • Spacebar
  • Start Menu
  • Switch Applications (Alt + Tab)
  • Tab
  • Undo (Ctrl + Z)

You can change the button actions as often as necessary while you determine which ones you prefer.
