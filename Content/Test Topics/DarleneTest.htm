Aircraft

The aircraft body manages all flight operations, including stabilization, navigation, camera operations, and monitoring. It includes a memory card slot for a Class 10 SD memory card (up to a maximum of 32 GB).It includes a memory card slot for a Class 10 SD memory card (up to a maximum of 64 GB).

The aircraft also includes a protector plate to cover the payload connector when the aircraft is not in use. If you are shipping the aircraft, the protector plate must be in place.

The aircraft also includes built-in cameras, including four downward-facing navigation cameras and one forward-facing EO/IR payload.

Landing gear legs

The four legs attach to the aircraft body and provide stable landing gear. If necessary, you can detach the legs, but they are generally left attached when storing the aircraft.

One leg includes a communications antenna. Use only the antennas supplied by Aeryon.

The four legs attach to the aircraft body and provide stable landing gear. All four legs have compasses and two legs have antennas. You must detach the legs before packing the aircraft.

Propulsion arms

Each of the four propulsion arms that attach to the aircraft body includes a motor and a fixed pitch propeller.

Each arm is color-coded to match the appropriate arm receptacle on the aircraft body.

Tablet and charger

Base station and charger

The base station and its antennas provide a long-distance communication link between the tablet and the aircraft. The base station includes a USB port, an Ethernet port, two long-range communications antennas, two short-range communications antennas, a built-in GPS antenna, a port to connect an external GPS antenna, and a tripod mount. An extra set of the four communications antennas is included with the base station.
