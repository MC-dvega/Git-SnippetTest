How The complexity of data analysis and reporting are done will depends on the objectives of your the DSC experiments. DSC data analysis can be simple or complex.

DSC data analysis process includes:

Subtraction of buffer-buffer reference baseline

The software automatically assigns and subtracts the matched buffer-buffer scan to correct for instrument and buffer baseline. YouThe user can select another buffer-buffer scan if desired.

Truncation of data range for curve fitting

You can This removes the starting and/or ending range of the thermogram, if desired for fitting.

Creation of integration baseline

The software determines the pre- and post-transition baseline positions, then creates an integration baseline chosen by the user (progress, spline, or line). YouThe user can adjust the baseline positions and choose the baseline type (progress, spline, or line) that is most suitable for yourtheir data. This integration baseline is used to determine the total area or enthalpy (ΔH) of the thermal denaturation.

Finding apparent Tm(s)

The software automatically finds the transition peak(s) of the thermogram. YouThe user can add additional transition(s) if desired, prior to fitting. These peaks are the Tm, known as the thermal transition (or “melting”) temperature, and is an indication of thermal stability – the higher the Tm, the more stable the protein.

Scan-rate normalization and concentration normalization
